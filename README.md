# Description

This Api is olny for get data directly from SAP

## Installation

1. Update .env.example file with Sql Server configuration

2. Rebuild node modules with command line

```
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
